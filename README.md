Penjelasan Program Tracker Langkah
Program ini adalah aplikasi pencatat langkah kaki harian yang dirancang untuk memantau aktivitas fisik pengguna selama satu minggu dengan target standar 10.000 langkah per hari. Program menggunakan struktur data DataHarian untuk menyimpan informasi nama hari, jumlah langkah, dan persentase capaian, kemudian melalui serangkaian fungsi seperti inputData() untuk pengumpulan data, calcProgress() untuk menghitung persentase pencapaian, serta berbagai fungsi analisis seperti calcTotalLangkah() dan jmlHariTercapai() untuk evaluasi statistik. Setelah data diinput, program menampilkan laporan harian yang mencakup progress persentase dan status pencapaian untuk setiap hari, dilanjutkan dengan ringkasan mingguan yang berisi total langkah, rata-rata harian, identifikasi hari terbaik dan terburuk, serta penilaian keseluruhan performa mingguan berdasarkan total kekurangan langkah, sehingga memberikan umpan balik komprehensif bagi pengguna untuk memantau konsistensi aktivitas fisik mereka.
